Fundamentals

APIs

Integrate and interact with other technologies available to the blockchain ecosystem. JavaScript frameworks like Ethers.js and Web3.js allow you to connect a browser frontend to the blockchain!

Nodes

A node is a computer that is connected to a network. In our case, each node is running the Aion kernel. Each node talks to other nodes on the network to do things like run applications, or confirm transactions.
